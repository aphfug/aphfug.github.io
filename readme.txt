mon site est une librairie de tout les petits programmes que j'ai fait pendant l'année
le site est organisé en 2 niveau :
-le premier est la page d'acceuil avec la page info et le css
-le deuxième est là où il y a toutes les pages avec chaque code (9 en python) et à chaque page correspond le fichier rar pout elecharger le programme

pour afficher le code :
-2 colonnes:
	-1 pour les lignes(qui sont écrites dans le code html une par une en fonction de la longeur du programme)
	-1 pour chaque lignes de code
-pour avoir les indentations j'ai créé 4 classes, une pour chaque niveau d'intentation (par exemple : tab1 = 20px et tab2 = 40px)

et il y a toujours un bouton télécharger pour télécharger le fichier rar où le programme est dedant.
