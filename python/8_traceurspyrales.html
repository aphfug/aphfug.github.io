Traceur de spyrales



Voici un de mes meilleurs programmes qui permait de tracer plein de polygones de plus en plus grands et qui s'organisent en spyrale. On peut choisir le nombre de cotés et l'angle entre chaque polygone

voici une image de ce que ça peut rendre (cotés : 3 | angle : 5)

from turtle import *
speed(0)
width(1)

def xAgone(lgr,n) :
for i in range(n):
forward(lgr)
right(360/n)

def figure(n) :
color((0,0,1))
for i in range(n+1) :
xAgone(i/n*900/cote, cote)
right(angle)
color((0+i/n,0,1-i/n/2))

cote=int(input("nombre de cotés ? (au moins 3)"))

if cote<3 :
print("non! j'ai dit plus de 3")
else :
angle = int(input("quel angle entre chaque polygone ?"))
figure(int(360/angle))

exitonclick()
